# frtpm
A simple RTP-MIDI to ALSA gateway


# how to build
* mkdir build
* cd build
* cmake ..
* make


# how to run
Just start it. It listens hardcoded on port 5004/5005 altough you can override 5004 using '-b' (5005 will be the next port).
